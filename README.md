# homebrew-window-manager

Homebrew tap for [EasySnaps Window Manager](https://easysnaps.org) — a macOS window manager with custom snap zones, smart profiles, and Dynamic Island-style tiling.

## Install

```sh
brew tap easysnaps/window-manager
brew install --cask easysnaps-window-manager
```

## Update

```sh
brew upgrade --cask easysnaps-window-manager
```

(The Window Manager also ships with built-in Sparkle updates, so new versions install automatically while the app is running.)

## Uninstall

```sh
brew uninstall --cask easysnaps-window-manager
```
